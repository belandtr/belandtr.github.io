Декілька кроків до твого успіху

Крок 1. Знайомство з WEB та розробкою сайтів.

Огляд та історія WEB та HTML. Поверхнево схема роботи Інтернет та WWW. HTML - це лише структура.
Інструментарій front-end розробника. Категорії контенту та базові елементи. Div, span. Заголовки. Семантичні елементи.
Практика.

Крок 2. Базові елементи побудови сайтів. Вступ CSS

Більше про блоки - div, span. Нумеровані та не-нумеровані списки. HTML entities. Синтаксис CSS. Прості селектори.
Практика.

Крок 3. Детальніше про CSS

Більш потужні селектори. Варіанти розміщення стилів. Приорітетність та успадкування стилей. Стилізація тексту.
Практика.

Крок 4. Продовження селекторів. Посилання. Хостинг

Види посилання. Псевдо-класи. Варіанти розміщення сайту (прості). Хостінг на github швидкий простий та безкоштовний варіант розміщення.
Практика.

Крок 5. Блоки. Основи позиціонування на сторінці

Модель позиціонування CSS. Базові параметри блоку. Спеціальні параметри блоку. Відступи та границі. Фон та градієнт.
Практика.

Крок 6. Позиціонування. Float.

Елементи, що плавають (float). CSS атрибути float та clear. Побудова макету з кількома колонками на базі float. Clearfix.
Практика.
